# Steganography Detection using CNN

## Description
This project detects hidden data in images using a Convolutional Neural Network (CNN).

## Features
- Image upload interface
- Real-time prediction (Clean / Stego)
- Accuracy visualization
- Flask web application

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Flask

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   python app.py

## Output
- Detects whether image contains hidden data
